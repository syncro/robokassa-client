# robokassa-client

Robokassa payments library

## Usage

Maven:

```
<dependency>
  <groupId>ru.robokasa</groupId>
  <artifactId>robokasa-client</artifactId>
  <version>0.0.1</version>
  <type>pom</type>
</dependency>
```

Gradle:

```
repositories {
    jcenter()
}

dependencies {
    compile 'ru.robokasa:robokasa-client:0.0.1'
}
```
